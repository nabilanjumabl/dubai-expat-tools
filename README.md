# Dubai Expat Tools 🇦🇪

Free toolkit for UAE expats — salary calculator, currency converter, gratuity calculator, VAT calculator, golden visa checker, rent estimator.

## Quick Deploy (5 minutes)

### 1. Push to GitHub
```bash
git init
git add .
git commit -m "Initial commit"
git branch -M main
git remote add origin https://github.com/YOUR_USERNAME/dubai-expat-tools.git
git push -u origin main
```

### 2. Deploy to Vercel (free)
1. Go to vercel.com → "Add New Project"
2. Import your GitHub repo
3. Click Deploy — done ✅
4. Get free domain: yoursite.vercel.app (or add custom domain)

### 3. Google Search Console
1. Go to search.google.com/search-console
2. Add property → URL prefix → https://yourdomain.com
3. Verify via HTML tag (add to layout.tsx metadata)
4. Submit sitemap: https://yourdomain.com/sitemap.xml
5. Wait 2-4 weeks for indexing

### 4. Google AdSense
1. Apply at adsense.google.com (need 20+ pages of content)
2. Add AdSense script to layout.tsx head
3. Wait 2-4 weeks for approval
4. UAE traffic RPM: $8–15 (premium audience)

## Local Development
```bash
npm install
npm run dev
# Open http://localhost:3000
```

## Tech Stack
- **Framework**: Next.js 15 (App Router)
- **Styling**: Custom CSS + Tailwind utilities
- **Hosting**: Vercel (free tier)
- **Currency API**: exchangerate-api.com (free tier)
- **SEO**: Built-in Next.js metadata + sitemap
